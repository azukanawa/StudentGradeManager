# 学生管理系统
# 如何运行它
1. 双击StudentGradeManger文件夹
2. 双击build文件夹
3. 双击Doc文件夹
4. 找到StudentGradeManager.exe程序，双击运行
# 文件结构
Doc文件夹主要用于存放 **.c** 后缀的文件， Lib文件夹则用于存放 **.h** 后缀的文件，build文件夹为CMake生成的构建文件，同时里面的Doc文件夹中的**测试数据.txt**是用于输入数据的 

# 注意事项
每个被读取的文件的学生数不应大于50， 其格式为：一行一条学生信息的记录，不同属性用空格隔开，输入数据中无总成绩记录。
  
  
| 属性名 | 姓名   | 学号   | 实验成绩(20%) | 半期成绩(30%) | 期末成绩(30%) | 项目成绩(20%) | 总评成绩（四舍五入） |
| ------ | ------ | ------ | ------------- | ------------- | ------------- | ------------- | -------------------- |
| 类型   | 字符串 | 字符串 | 整数          | 整数          | 整数          | 整数          | 整数                 |

测试数据应通过build文件中的Doc文件夹中的**测试数据.txt**输入  

请按照要求的格式输入数据
